# GPT 코드 리뷰 프롬프트

Target-UP 프로젝트의 코드 리뷰를 담당합니다.

## 역할
- DB 스키마 리뷰
- API 설계 리뷰
- 보안 취약점 검토
- 코드 품질 검토
- 테스트 케이스 작성

---

## 리뷰 요청 1: DB 스키마 리뷰

아래 PostgreSQL 스키마를 리뷰해주세요.

### 검토 항목

1. **정규화/비정규화 적절성**
   - 불필요한 중복 데이터 있는지
   - 조회 성능을 위한 비정규화가 적절한지

2. **인덱스 설계**
   - 누락된 인덱스
   - 불필요한 인덱스
   - 복합 인덱스 필요 여부

3. **데이터 타입**
   - UUID vs BIGSERIAL 선택 적절성
   - VARCHAR 사이즈 적절성
   - JSONB 사용 적절성

4. **제약 조건**
   - 누락된 NOT NULL
   - CHECK 제약 추가 필요 여부
   - 외래 키 ON DELETE 정책

5. **보안**
   - 민감 데이터 암호화 필요 여부
   - 개인정보 관련 컬럼 처리

6. **확장성**
   - 파티셔닝 전략 적절성
   - 향후 확장 고려

7. **한국 규제 준수**
   - 개인정보보호법 관련
   - 발신번호 사전등록제 관련
   - 수신거부 처리 관련

### 출력 형식

```markdown
## 스키마 리뷰 결과

### ✅ 잘된 점
- ...

### ⚠️ 개선 필요
| 테이블 | 이슈 | 권장 수정 |
|--------|------|----------|
| ... | ... | ... |

### 🔴 필수 수정
- ...

### 💡 추가 제안
- ...
```
