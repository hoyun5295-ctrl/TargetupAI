## 2. 기능별 상세 설계

### 🔵 기능 1 — POS 실시간 데이터 브릿지

**목적:** 사장님이 전단 만들 때 "어떤 상품을 밀어야 할지" AI가 추천
**데이터:** flyer_pos_sales (최근 7일 집계), flyer_pos_inventory (재고/유통기한)
**API:**
- `GET /api/flyer/pos/top-products?days=7&limit=20` → TOP 판매 상품
- `GET /api/flyer/pos/overstock` → 재고 과다 (30일 판매량 < 현재 재고/2)
- `GET /api/flyer/pos/expiring-soon?days=3` → 유통기한 임박
- `GET /api/flyer/pos/high-margin?limit=10` → 마진율 높은 상품
**UI:** 전단 생성 화면 상단에 "POS 추천 상품" 탭 → 클릭 한 번으로 전단에 추가
**난이도:** 중 (POS Agent 선행 필요)

### 🔵 기능 2 — POS 고객 DB 자동 싱크

**목적:** 엑셀 업로드 제거. POS 회원가입 → 자동 flyer_customers INSERT
**싱크 주기:** Agent가 10분마다 증분 푸시
**매칭 키:** pos_member_id + phone
**UPSERT 로직:**
```sql
INSERT INTO flyer_customers (...)
ON CONFLICT (company_id, phone)
DO UPDATE SET name=EXCLUDED.name, pos_grade=EXCLUDED.pos_grade, ...
```
**수신동의 처리:** POS에 수신동의 플래그 있으면 그대로 반영. 없으면 default false로 저장하고 사장님이 직접 체크
**난이도:** 중

### 🔵 기능 3 — ROI 역추적 ⭐

**목적:** "이 전단 보내고 실제 매출 얼마 올랐는가"를 숫자로 증명
**처리 흐름:**
```
전단 발송 (flyer_campaigns.sent_at = T0)
  ↓
T0 ~ T0+7일 동안 flyer_pos_sales 수집
  ↓
sent_recipients(수신자 phone 목록) ∩ flyer_pos_sales.customer_id
  ↓
매칭된 고객의 구매 총액 → flyer_campaigns.roi_revenue 업데이트
```
**리포트 UI:**
- 수신자 10,000명 중 **842명이 실제 방문 구매**
- 총 매출 **₩12,340,000** (발송비 ₩80,000 대비 ROI **154배**)
- 상품별 판매 TOP 10 (전단에 실린 상품 강조)
**난이도:** 중 (집계 배치 필요)

### 🔵 기능 4 — 주간/월간 정기 자동발행

**재활용:** 한줄로 auto-campaign-worker.ts 패턴 → flyer-auto-worker.ts로 복제
**DB:** flyer_auto_campaigns 테이블 (한줄로 auto_campaigns 미참조)
**플로우:**
```
매주 목요일 오전 9시 → POS TOP 20 상품 자동 선정 →
AI가 전단 문구 생성 → 사장님 슬랙/카톡 알림 →
승인 클릭 시 오전 11시 발송
```
**승인 없이 자동발송 옵션도 제공** (신뢰 구축 후)
**난이도:** 중 (한줄로 로직 복제 + 전단AI 특화)

### 🔵 기능 7 — 전단 템플릿 다양화

**현재:** 3종 템플릿
**확장:** 업종별 + 시즌별 + 테마별 템플릿 라이브러리
- 마트 일반/식자재마트/로컬슈퍼
- 정육/수산/농산/청과
- 명절(설/추석)/개업/창고대방출/신상입고
- 시즌: 봄 딸기축제, 여름 수박/삼계탕, 가을 전어, 겨울 귤/굴
**난이도:** 낮음 (HTML 템플릿 추가 + DB 카테고리)

### 🔵 기능 8 — QR 체크인 쿠폰

**목적:** "전단 → 실제 매장 방문" 측정
**플로우:**
```
전단 하단 QR → 스캔 시 /q/:code 페이지 → "쿠폰 사용" 버튼 →
POST /api/flyer/coupons/redeem → 쿠폰 사용 처리 +
매장 POS에서 동일 쿠폰 코드 입력 시 할인 적용
```
**DB:** flyer_coupons, flyer_coupon_redemptions
**난이도:** 중 (POS와 쿠폰 코드 연동 방식 정의 필요)

### 🔵 기능 9 — RFM 자동 세분화

**Recency:** 최근 방문일 (days)
**Frequency:** 방문 횟수 (최근 90일)
**Monetary:** 총 구매액
**세그먼트:**
- Champion: R<7, F>=5, M>=상위10%
- Loyal: R<14, F>=3
- New: R<30, F=1
- At Risk: 30<=R<60
- Lost: R>=60
- Whale: M>=상위5%

**업데이트 주기:** Agent가 판매 데이터 싱크할 때마다 해당 고객 재계산
**UI:** 전단 발송 시 "세그먼트로 타겟" 드롭다운
**난이도:** 중

### 🔵 기능 10 — 생일자/기념일 자동 전단

**플로우:**
```
매일 오전 9시 → flyer_customers WHERE EXTRACT(MONTH FROM birth_date) = 이번달
→ AI 개인화 메시지 + 특별 쿠폰 포함 전단 자동 생성 → 발송
```
**템플릿:** "○○님, 생일 축하드립니다! 이번 주 매장 방문 시 5,000원 쿠폰 증정 🎂"
**난이도:** 낮음 (자동발송 엔진 재활용)

### 🔵 기능 11 — 이탈 고객 윈백

**타겟:** `last_purchase_at < now() - 60일` AND `total_purchase_amount >= 회사 평균의 2배`
**메시지:** "오랜만이에요! 특별 할인 쿠폰을 준비했습니다"
**난이도:** 낮음

### 🔵 기능 12 — 장바구니 기반 추천

**데이터:** flyer_pos_sales에서 동일 receipt_no의 상품 동시구매 쌍 집계
**예:** "삼겹살 구매자의 67%가 상추 동시구매" → 상추 할인 전단을 삼겹살 구매자에게
**난이도:** 중상 (Apriori 알고리즘 or 간단한 co-occurrence)

### 🔵 기능 13 — 카카오 브랜드메시지 배포

**재활용:** CT-12 brand-message.ts
**주의:** 한줄로 CT-12는 한줄로 브랜드 전용 → flyer 전용 래퍼 필요 or CT-12를 shared로 승격
**난이도:** 낮음

### 🔵 기능 14 — A/B 테스트

**플로우:**
```
전단 생성 시 "A/B 테스트" 토글 →
헤드카피 2종 or 썸네일 2종 입력 →
수신자 랜덤 50/50 분할 →
각 버전별 클릭률/ROI 비교 리포트
```
**난이도:** 중

### 🔵 기능 15 — 카탈로그 DB (재사용 자산화)

**목적:** 한 번 만든 상품은 다음 전단 제작 시 재사용
**데이터:** flyer_catalog 테이블
**UI:** 전단 생성 화면 좌측 "내 상품" 사이드바 → 드래그 앤 드롭
**POS 연동 시:** pos_product_code로 자동 연결 → 가격/재고 실시간 반영
**난이도:** 낮음 (⭐ Phase A 1순위)

### 🔵 기능 16 — 매장 직원 간편 모드

**컨셉:** 사장님이 자리에 없어도 알바생이 쓸 수 있는 3단계 UI
```
1. 사진 찍기 (상품 스마트폰 카메라)
2. 가격 입력 (숫자패드)
3. [전송] 버튼 → 오늘 타겟 고객(기본 세그먼트)에 자동 발송
```
**권한:** flyer_staff 역할 신설, 세그먼트 선택 불가 (기본 세그먼트만)
**난이도:** 중 (모바일 우선 UI)

### 🔵 기능 17 — 인쇄 PDF 내보내기

**라이브러리:** puppeteer or playwright로 공개 페이지 → PDF 변환
**레이아웃:** A4 / A3 / A2 / 반페이지 / 양면
**난이도:** 낮음

### 🔵 기능 18 — 재고 연동 품절 표기

**로직:** 공개 페이지 렌더링 시 flyer_pos_inventory 실시간 조회 → current_stock=0 상품에 "품절" 오버레이
**캐시:** 5분 캐시 (과도한 DB 조회 방지)
**난이도:** 낮음

## 5. 영업 메시지 (마트 사장님 대상)

**핵심 소구점 3줄:**
1. **"월 15만원으로 전단지 + 문자 + 효과 측정까지"** — 전단 외주 비용(월 30~50만원) 대비 절반 이하
2. **"POS랑 연결하면 판매 데이터로 전단 자동 추천"** — 사장님이 고민 안 해도 됨
3. **"전단 발송 후 실제 매출 증가 숫자로 확인"** — 효과 눈으로 봄

**가격 구조:**
- 기본: 15만원/월 (전단 무제한, 고객DB 무제한, POS 연동 포함)
- 발송비 별도: SMS 9원 / LMS 29원 / MMS 80원 (한줄로 단가와 동일)
- 첫 달 무료 체험 (마트 사장님 진입 장벽 낮춤)
