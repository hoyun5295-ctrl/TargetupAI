## 4) 카테고리별 버그 가이드

### 4-1. 전단지 페이지 관련
- **체크 포인트:** 미리보기 ↔ 발행 ↔ 공개 페이지 3곳 동기화 확인
- **과거 사고:** 미리보기와 공개 페이지의 상품 정렬 순서 불일치

### 4-2. SMS/LMS/MMS 발송 관련
- **체크 포인트:** 한줄로 발송 파이프라인(campaigns.ts) 재활용 중이므로 **한줄로 수정 시 전단AI에도 영향**
- **방어:** D112 이후 flyer_campaigns 테이블로 완전 분리 → 한줄로 campaigns.ts 수정 영향 0

### 4-3. POS Agent 연동
- **체크 포인트:** POS 종류별 문자셋(EUC-KR/CP949/UTF-8), 날짜 포맷, 재고 단위
- **로그 확인:** Agent 푸시 로그 → flyer-backend 수신 로그 → DB INSERT 확인

### 4-4. 슈퍼관리자 서비스 스위처 (D112~)
- **체크 포인트:** 전단AI 모드에서 한줄로 회사 조회/수정 시도 시 401 반환 확인
- **방어:** URL 경로(/flyer/*) + JWT service 필드 이중 체크
